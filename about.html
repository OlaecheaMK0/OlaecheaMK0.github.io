About me

I'm a Computer Science student at the University of Central Florida, focused on build-and-ship work — the kind where I can see what I made at the end of the day.

At UCF, my coursework this year includes CAP 4611 (Machine Learning), CIS 3360 (Cybersecurity), and the CS 1 / CS 2 sequence — the theory behind the projects I build on the side.

Outside class I build trend-driven personal projects — whatever technology is currently pulling me in. One recent build uses OpenClaw to run an LLM agent that classifies Plaid API transactions into accounting periods in a spreadsheet and surfaces budgeting recommendations. I used the same framework against the Canvas API to pair every Logseq daily journal entry with my UCF assignments, due dates, and class information. My projects are born from my inconveniences.

I'm comfortable in Python, C, C++, Java, and JavaScript (plus HTML and CSS when a project needs a front-end). My data work leans on pandas, scikit-learn, and techniques like k-means clustering. Day-to-day I live in Git, OrbStack, and Logseq. Outside of code: I speak English and Spanish, and I'm casually learning Japanese.
